# UniSphere - Smart Campus Management System

## Overview
UniSphere is an Object-Oriented Programming (OOP) based Smart Campus Management System developed in C++.

## Features
- Student Management
- Teacher Management
- Course Management
- Department Management
- Attendance Management
- Result Management
- Fee Management
- Timetable Management
- Authentication System
- File Handling

## Technologies
- C++
- Object-Oriented Programming
- Git
- GitHub

## Status
Currently under development.
